Hismith Piupiu Emulator

Flash an ESP32 relay board so it advertises as a Hismith Piupiu over BLE and drives a pump from the documented squirt command.

2. Wire up a USB-TTL adapter

The RobotDyn ESP32R4 has no USB port — it needs a USB-to-TTL serial adapter (a CP2102 is the usual choice), set to 3.3V.

  • 3.3V → board 3.3V
  • GND → board GND
  • TX → board RX
  • RX → board TX

Remove the jumper next to the ESP32 first, and never have the mains/DC supply connected at the same time as the adapter's 3.3V.

3. Put it in bootloader mode, then flash

There is no auto-reset on a 4-wire hookup, so the board cannot be put into bootloader mode for you. Do it by hand before clicking flash: hold Prog → press and release RST → release Prog. Then click flash and pick the adapter's serial port.

This browser cannot flash. Web Serial is required — use Chrome or Edge on desktop. Flashing requires a secure (HTTPS) page.

After flashing

  • The board advertises as Hismith Piupiu, service ffe5, write characteristic ffe9.
  • Writing cc 0b 01 0c closes the relay; it stays closed while the commands keep coming and releases shortly after they stop.
  • The default build drives GPIO25 (Relay 1 on a RobotDyn ESP32R4) as active-high. If your relay does not click, the pinout differs — see the self-test build in the README.
